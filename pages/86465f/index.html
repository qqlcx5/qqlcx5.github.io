permissions 0755 问题解决

# permissions 0755 问题解决

# 前言

我最近京东买了电脑配置搞了一台黑苹果，成本4k左右配置，可以媲美满配置的苹果笔记本，苹果的硬件利润真的多。

# 报错分析🔍

我在这台黑电脑上安装git后，发现克隆不了仓库，如图下问题。

简单翻译一下这句报错信息：

“/Users/another/.ssh/id_rsa”的权限0755太开放。
要求您的私钥文件不能被其他人访问。
此私钥将被忽略。
Load key“/Users/another/.ssh/id_rsa”：权限错误
git@github.com：权限被拒绝（公钥）。
致命：无法从远程存储库读取。
请确保您具有正确的访问权限
存储库存在。

# 解决方案

出现该问题主要是因为你的ssh中的私钥权限导致的。

chmod -R 700 ~/.ssh/*

参数

  • 700 只有拥有者有读、写、执行权限。
  • -R 递归改文件内的权限。
